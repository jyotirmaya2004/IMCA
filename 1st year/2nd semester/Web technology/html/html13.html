All Text Formatting Tags in HTML

1. Bold Text: This is bold text using the <b> tag.

2. Strong Text: This is strong text using the <strong> tag (semantically important).

3. Italic Text: This is italic text using the <i> tag.

4. Emphasized Text: This is emphasized text using the <em> tag (semantically important).

5. Underlined Text: This is underlined text using the <u> tag.

6. Strikethrough Text: This is strikethrough text using the <s> tag.

7. Deleted Text: This is deleted text using the <del> tag (semantically important).

8. Superscript Text: This is superscript text using the <sup> tag (e.g., x2).

9. Subscript Text: This is subscript text using the <sub> tag (e.g., H2O).

10. Highlighted Text: This is highlighted text using the <mark> tag.

11. Small Text: This is small text using the <small> tag.

12. Inserted Text: This is inserted text using the <ins> tag.

13. Code Text: This is code text using the <code> tag.

14. Preformatted Text: This is preformatted text using the <pre> tag:

            This is preformatted text.
            It preserves spaces and line breaks.

15. Blockquote: This is a blockquote using the <blockquote> tag:

"This is a blockquote. It is used to represent a section of quoted content."

16. Abbreviation: This is an abbreviation: HTML using the <abbr> tag.

17. Address: This is an address using the <address> tag:

Written by John Doe.
Visit us at:
Example.com
Box 123, Disneyland
USA

18. Inline Quotation: This is an inline quotation: This is a short quote using the <q> tag.

19. Time: This is a time element: October 15, 2023 using the <time> tag.
